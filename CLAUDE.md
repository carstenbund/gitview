# use GitView Commit Format
```
<type>(<scope>): <subject>

<body>

Storyline: [STATUS:CATEGORY] Initiative Title
```

**Types:** feat, fix, refactor, docs, test, chore, ci

**Status:** new, continued, completed, stalled

**Categories:** feature, refactor, bugfix, debt, infrastructure, docs, test

**Rules:** Keep storyline titles consistent across related commits. Use `completed` when finishing.
